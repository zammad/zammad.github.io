# zammad.github.io
[Helm](https://helm.sh) repo for [Zammad](https://zammad.org) Helm chart which can be installed on [Kubernetes](https://kubernetes.io)

Add the repo via:

```console
helm repo add zammad https://zammad.github.io/zammad-helm
```
